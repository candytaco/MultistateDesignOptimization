Class EntropyWeightsMixedSimilarity

Mixes two similarity measures based on the entropy of the target sequence at each position.

Methods

__init__(self, SM1:SimilarityMeasure, SM2:SimilarityMeasure, targetFrequencies:numpy.array)

Constructor. SM1 is given more weight with increasing entropy, and SM2 is given more weight with lower entropy. Empty constructors for SM1 and SM2 can be used, since EntropyWeightsMixedSimilarity will handle the target frequency assignments.
