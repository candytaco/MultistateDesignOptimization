Class SimilarityMeasure

Overview

An abstract class/interface defining the interface of a method for measuring the similarity between two distributions, in this case specifically for amino acid residue frequencies

Fields

Field Type Description
targetFrequencies float[][] as numpy.array The target residue frequencies to which to compare a calculated frequency set. The first index is position, and the second index is residue by alphabetical order of single-letter codes.

abstract getSimilarityMeasure(self, expFrequencies) -> float

Sees how similar an experimental frequency set is to the target/natural set. Inheriting subclasses are expected to provide an overriding implementation of this method.

Parameter Type Optional Description
